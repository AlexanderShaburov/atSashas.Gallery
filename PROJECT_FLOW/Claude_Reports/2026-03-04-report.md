# Report: Unified Renderer + Advanced Frame Editing — 2026-03-04

## Summary
Implemented 6-phase unified gallery renderer with frame edit capabilities.

## Phase 1: Renderer Unification
- **New types** in `entities/block/block.types.ts`: `RenderContext`, `LayoutGeometry`, `SlotFrameStyle`, `CaptionStyle`
- Extended `GalleryBlock` with optional `geometry` and `captionStyle` fields
- Extended `GalleryArtItem` with optional `frameStyle` and `captionStyle` fields
- **New shared component** `shared/ui/GalleryRenderer/` — unified position-class CSS renderer used by both admin and public
  - `GalleryRenderer.tsx` — main component with slot rendering, grid style computation, frame style application
  - `GalleryRenderer.css` — merged layout rules from `block-gallery.core.css` and `Gallery.css` with `.gr` namespace
  - `ResizeHandle.tsx` — drag-to-resize split ratio between slots
  - `SlotCropOverlay.tsx` — drag-to-reposition images, scroll-to-zoom
  - `CaptionStylePopover.tsx` — caption style editor (font size, weight, color, position)
- Extended `ArtPicture` with `imgStyle` prop for object-position/transform
- **Migrated** `GalleryComponent.tsx` (admin) to delegate rendering to `<GalleryRenderer>`
- **Migrated** `ImageComponent.tsx` (public) to use `<GalleryRenderer context="public">`

## Phase 2: Frame Edit Mode Toggle
- Added `'frameEdit'` to `ToolKey` union in toolbar types
- Added `isFrameEditMode` / `onToggleFrameEdit` to `ToolbarCtx`
- Added `FrameEditButton` toolbar element with active state styling
- Registered in `TOOL_REGISTRY` — shows for gallery blocks only
- Added `isFrameEditMode`, `toggleFrameEdit`, `updateGeometry`, `updateSlotFrameStyle` to `BlockEditorSession` context
- Wired through `SingleBlockEditor` → `GalleryComponent` → `GalleryRenderer`

## Phase 3: Frame Resize
- `LayoutGeometry` model supports `splitRatio` and `smallColumnRatio`
- `ResizeHandle` uses pointer events for drag, computes ratio relative to container
- `GalleryRenderer` applies dynamic `grid-template-columns` via inline style when geometry is set

## Phase 4: Image Position/Offset
- `SlotFrameStyle.objectPositionX/Y` (0-100%) applied as `object-position` on `<img>`
- `SlotCropOverlay` drag handler computes position delta relative to container

## Phase 5: Image Crop/Zoom
- `SlotFrameStyle.scale` (1.0-3.0) applied as `transform: scale()` on `<img>`
- Scroll wheel on crop overlay adjusts scale
- Reset button clears all frame style customizations

## Phase 6: Caption Editing
- `CaptionStyle` model with fontSize, fontWeight, color, position, background
- `CaptionStylePopover` with preset dropdowns and color picker
- Styles applied to public block caption via `captionPositionStyle()`

All module-level singletons (storage paths, repos, `UserRepository`) resolve to the temp vault. Cleanup via `pytest_sessionfinish`.

**Auth bypass**: Admin endpoint tests use `app.dependency_overrides[get_current_user]` to return a mock `User` without real credentials.
